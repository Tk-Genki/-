# 入浴タイムスタンプ記録アプリ 試作版

高齢者施設職員がスマホで片手操作しやすいことを重視した、Google Apps Script ベースの簡易記録アプリです。

## 構成

- `Code.gs`
  - 画面表示
  - スプレッドシート保存
  - 利用者マスタの試作データ
- `Index.html`
  - 入力画面本体
- `Stylesheet.html`
  - スマホ向け UI
- `JavaScript.html`
  - ボタン押下時の自動時刻記録と保存処理

## できること

- 利用者IDを選択
- 担当者名を入力
- `脱衣開始` `洗体開始` `入浴開始` `終了` を大きいボタンで記録
- ボタンを押した瞬間の時刻を自動保存
- 1回の入浴ごとに `session_id` を自動付与
- 同じ入浴内の工程を同一 `session_id` で紐付け
- 曜日と時間帯を自動保存
- Google スプレッドシートへ1件ずつ追記
